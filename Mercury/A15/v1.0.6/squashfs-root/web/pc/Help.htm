  • SSID-3
  • 工作在5GHz频段，对应路由器5G主人网络，可在高级设置-Wi-Fi设置-主人网络界面进行设置。
  • SSID-4
  • 工作在5GHz频段，对应路由器5G访客网络，可在高级设置-Wi-Fi设置-访客网络界面进行设置。
  • SSID-5
  • 工作在2.4GHz频段，根据移动定制要求，只接受移动家庭管理平台远程设置，不允许本地设置。
  • SSID-6

  • 基本设置
  • 自动获得IP地址上网方式
  • 路由器自动从运营商获得IPv6地址进行上网。请注意更改WAN口连接类型时，局域网终端可能会断开几秒后自动重连。
  • IPv6地址、网关、DNS服务器
  • 路由器从运营商获取的上网参数。当前路由器仅支持64位子网前缀长度。
  • WAN口高级参数
  • IPv6地址获取协议
  • 默认自动，也可以根据需要进行相应修改。如果选择DHCPv6，则直接由运营商动态分配一个IPv6地址；如果选择SLAAC，则由路由器根据路由通告自动生成IPv6地址。
  • 前缀授权
  • 开启此功能后，路由器将自动从运营商获取一个IPv6地址前缀，该前缀用于为局域网中设备生成IPv6地址。此时"局域网设置" -> "IPv6地址前缀"将不允许手动修改。开启此功能需要运营商支持，否则将无法获取IPv6地址前缀。请根据实际情况正确选择是否开启。
  • DNS服务器
  • 运营商会自动分配DNS服务器，如有需要也可以手动设置。
  • 基本设置
  • 固定IP地址
  • 使用运营商提供的固定IPv6地址进行上网。请注意更改WAN口连接类型时，局域网终端可能会断开几秒后自动重连。
  • IPv6地址、子网前缀长度、网关、DNS服务器
  • 固定IP地址上网方式需要填写的上网参数，由您的运营商提供。当前路由器仅支持64位子网前缀长度。
  • WAN口高级参数
  • 数据包MTU
  • 数据包的最大传输单元，可设置范围为576 ～ 1500。
  • 基本设置
  • 宽带拨号上网
  • 使用运营商提供的宽带帐号和密码进行上网的方式。请注意更改WAN口连接类型时，局域网终端可能会断开几秒后自动重连。
  • IPv6宽带帐号/IPv6宽带密码
  • 运营商提供的宽带帐号和密码。
  • 复用IPv4拨号链路
  • 当IPv4上网方式也为宽带拨号上网时，可以选择是否开启此功能。开启此功能后，IPv6将使用IPv4账号密码进行拨号，不需要手动输入IPv6宽带账号及密码。请注意开启此功能需要运营商支持，请根据实际情况正确选择是否开启。
  • IPv6地址、DNS服务器
  • 当前路由器获取的上网参数。当前路由器仅支持64位子网前缀长度。
  • WAN口高级参数
  • IPv6地址获取协议
  • 默认自动，也可以根据需要进行相应修改。如果选择DHCPv6，则直接由运营商动态分配一个IPv6地址；如果选择SLAAC，则由路由器根据路由通告自动生成IPv6地址；如果选择固定IP，则使用运营商提供的固定IPv6地址进行上网。
  • 前缀授权
  • 当IPv6地址获取协议为自动、DHCPv6或者SLAAC时，可以选择是否开启前缀授权功能。开启此功能后，路由器将自动从运营商获取一个IPv6地址前缀，该前缀用于为局域网中设备生成IPv6地址。此时"局域网设置" -> "IPv6地址前缀"将不允许手动修改。开启此功能需要运营商支持，否则将无法获取IPv6地址前缀。请根据实际情况正确选择是否开启。
  • 桥模式
  • 当选择此模式时，路由器IPv6工作在桥模式。连接到路由器上的终端将由路由器上联设备分配IPv6地址及提供路由功能。
  • 局域网设置
  • 主机配置DHCPv6
  • 路由器将自动为局域网中设备分配IPv6地址、DNS服务器等上网参数。
  • IPv6地址前缀
  • 请输入运营商提供的IPv6地址前缀。只有正确输入IPv6地址前缀后，局域网中设备才能获取到正确的IPv6地址。当高级设置中开启前缀授权时，路由器将自动从上级设备获取IPv6地址前缀，不需要手动输入。当前路由器只支持64位IPv6地址前缀。

  • 自动分配的IP的有效时间，超过该时间后局域网内的设备将重新获取IP。

  • 路由器对局域网的IPv6地址，由路由器根据IPv6地址前缀结合自身MAC地址自动生成。

  • 主机配置SLAAC

  • 设置路由器为局域网中设备分配DNS服务器地址的方式。不同终端支持的情况可能不同，需要根据实际使用情况正确配置。

  • e-Link协议开关
  • 如果开启，则家庭网关通过e-Link协议对路由器进行配置，勾选“启用与智能网关的无线配置同步”开启相关配置同步功能。
  • e-Link功能重要说明
    • e-Link开关涉及到设备工作模式（路由器/桥）切换。设备出厂模式下处于路由模式，且e-Link协议开关默认开启。
    • 设备WAN口与智能网关LAN口相连，且e-Link协议开关开启时，若设备处于路由模式，则自动同步为桥模式。（备注：设备上网方式为宽带拨号上网时除外，此时e-Link协议强制关闭。）
    • 当设备经e-Link协议同步为桥模式时：若勾选e-Link协议开关，设备自动同步网关下发的配置信息；若取消勾选e-Link协议开关，则设备不再同步网关下发的配置信息，但仍保持桥模式。
    • 当设备经e-Link同步为桥模式时，若要恢复路由模式，有如下两种方法。
      • 方法一：
      • （1）断开设备WAN口与智能家庭网关连接；
      • （2）恢复出厂设置；
      • （3）登陆管理页面，进入路由设置-e-Link设置界面，取消勾选“启用与智能网关的无线配置同步”开关。
      • 需要注意的是，在（2）、（3）完成之前，需要保持设备WAN口与智能家庭网关之间连接断开。
      • 方法二：
      • 关闭e-Link同步开关，进入路由设置-路由/桥模式切换 界面，选择设备工作模式为路由模式。
